Sass

This boilerplate uses latest version of Sass. The project structure is based on Smacss styling guideline.

Gulp

The latest Gulp version is used as the build system. You can tweak the packaged Gulp tasks as you see fit.

Bootstrap 3

This Kickstarter ships with Bootstrap 3.3.7 via CDN. For your convenience, don't change it unless you know what you're doing.
